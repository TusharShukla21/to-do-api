CHAPTER 01

Why does Git exist?

Before Git, saving versions of code looked like this:

# your folder, in 2003
app.py
app_v2.py
app_v2_FINAL.py
app_v2_FINAL_real.py
app_DONT_DELETE.py    # which one is real?
Git is a time machine for your code. Every save becomes a labelled checkpoint you can jump back to — forever.
CHAPTER 02

Git vs GitHub

Two completely different things. Don't confuse them.

TOOL

Git

  • Lives on your laptop
  • Works offline
  • Tracks your changes
  • You type commands
WEBSITE

GitHub

  • Lives on the internet
  • Needs Wi-Fi
  • Stores Git projects
  • You click buttons
Git is the engine. GitHub is the garage. Engine runs fine without the garage — but the garage is where you store, share, and let others see it.
CHAPTER 03

The 3 Zones

Every file in Git lives in one of three zones. Learn this and every command makes sense.

✎
Working
you edit here
→ git add
☐
Staging
ready to save
→ git commit
●
Repository
saved forever
You edit a file → you stage it (pick what to save) → you commit (take the snapshot). That's it. That's Git.

CHAPTER 05

Branches

A branch is a parallel copy of your project where you can experiment safely.

main = the real, working project. feature branches = where you try new things. Merge back into main only when it works. 
git switch -c feature/login   # create + switch
# ...write code, commit...
git switch main                # go back to main
git merge feature/login        # bring changes in

CLICK TO REVEAL
Internet is off. Can I still commit?

Yes. Commits are local. You only need internet for push and pull.

What's the difference between git add and git commit?

add picks which changes to include. commit actually saves them as a snapshot.

I deleted a file by mistake. Is it gone forever?

No. If it was committed before, you can restore it with git restore filename. Git remembers everything.

Why use a branch instead of editing main directly?

Branches isolate risk. If your experiment fails, you throw the branch away and main stays clean.

What does git push do?

Uploads your committed snapshots from your laptop to GitHub.
